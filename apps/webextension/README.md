# `react-chrome-extension-apps`

## `webextension`

### Motivation

I wanted to prove out following 

 - [`parcel` `webextension` recepie](https://parceljs.org/recipes/web-extension/) 
 - For a [ReactJS application](https://parceljs.org/recipes/react/)


### Setup

#### First time

```bash
$> npm run setup
```

#### Dependencies

```bash
$> npm install
```

### `dev`

```bash
$> npm run dev
```

### Build

#### `dist/`

```bash
$> npm run dist
```
